## 소개

OneLLM은 다양한 AI 언어 모델들을 하나의 통합된 인터페이스에서 사용할 수 있게 해주는 웹 애플리케이션입니다. Samsung의 OneUI 디자인 철학을 바탕으로 직관적이고 사용하기 쉬운 인터페이스를 제공합니다.

## 주요 기능

- 다양한 LLM 모델 지원
- OneUI 기반의 모던한 디자인
- 실시간 채팅 인터페이스
- 반응형 디자인
- 다크 모드 지원
- 안전한 API 키 관리

## 시작하기

```bash
# 저장소 클론
git clone https://github.com/sioaeko/OneLLM.git

# 디렉토리 이동
cd OneLLM

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 기술 스택

- React 18
- Vite
- TailwindCSS
- TypeScript
- Netlify
